# OOP Examples (Simple Cases)

This folder contains short, runnable OOP examples based on the guide.

## Files by Case

- `oop_01.py` - Classes and objects
- `oop_02.py` - Instance, class, and static methods
- `oop_03.py` - Encapsulation with `@property`
- `oop_04.py` - Inheritance and `super()`
- `oop_05.py` - Multiple inheritance and MRO
- `oop_06.py` - Polymorphism and duck typing
- `oop_07.py` - Method overriding
- `oop_08.py` - Abstraction with `ABC`
- `oop_09.py` - Magic/dunder methods
- `oop_10.py` - Inner classes
- `oop_11.py` - Class vs instance attributes
- `oop_12.py` - Composition (HAS-A relationship)
- `oop_13.py` - Employee class model
- `oop_14.py` - Employee manager logic
- `oop_15.py` - Employee menu frontend (interactive)
- `oop_16.py` - Microwave OOP example
- `oop_17.py` - Dataclass basics
- `oop_18.py` - Aggregation (student and courses)
- `oop_19.py` - Mixins for reusable behavior
- `oop_20.py` - Enum-based object state
- `oop_21.py` - Context manager class (`__enter__` / `__exit__`)
- `oop_22.py` - Strategy pattern with `Protocol`
- `oop_23.py` - OOP form using validations from `utils.py`
- `oop_24.py` - Interactive CRUD app using validations from `utils.py`

## How to Run

From the project root:

```bash
python ITSE-1003/Examples/oop_01.py
```

Replace `oop_01.py` with any other file from this list.
