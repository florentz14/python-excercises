# 13_Queues

Queue data structure (FIFO). This folder covers fundamental queue operations, circular queues,
deque implementations, priority queues, BFS patterns, and classic queue interview problems.

## Files

| File | Content |
| --- | --- |
| `01_queue_with_list.py` | Queue implemented with Python list |
| `02_deque.py` | Queue operations using `collections.deque` |
| `03_queue_class_deque.py` | Queue class built on top of `deque` |
| `04_printer_queue.py` | Printer queue simulation |
| `05_customer_service_priority.py` | Customer service queue with VIP rules |
| `06_circular_queue_round_robin.py` | Round-robin scheduler with circular queue |
| `07_bfs_graph.py` | BFS graph traversal with queue |
| `08_call_center.py` | Call center waiting queue simulation |
| `09_priority_queue_heapq.py` | Priority queue with `heapq` |
| `10_priority_queue_class.py` | Custom priority queue class |
| `11_monotonic_queue.py` | Monotonic queue pattern |
| `12_sliding_window_maximum.py` | Sliding Window Maximum (LeetCode 239) |
| `13_bfs_shortest_path.py` | Shortest path in unweighted graph using BFS |
| `14_multi_source_bfs.py` | Multi-source BFS pattern |
| `15_task_scheduler_queue.py` | Task scheduler with cooldown logic |
| `16_linear_queue.py` | Array-based linear queue ADT |
| `17_circular_queue.py` | Circular queue ADT with wrap-around indexing |
| `18_deque_doubly_linked_list.py` | Deque built using a doubly linked list |

| `20_number_of_recent_calls.py` | Number of Recent Calls (queue time window) |
| `21_dota2_senate_queue_simulation.py` | Dota2 Senate queue simulation |

| `24_time_needed_to_buy_tickets.py` | Ticket buying time simulation |
| `25_reveal_cards_in_increasing_order.py` | Reveal Cards in Increasing Order |
