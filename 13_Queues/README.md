# 13_Queues

Queue data structure (FIFO). Covers basic queues, priority queues, monotonic queues, BFS algorithms, and task scheduling.

## Files

| File | Content |
|------|---------|
| `01_list.py` | Queue using list |
| `02_deque.py` | Queue using deque |
| `03_queue_class_deque.py` | Queue as class using deque |
| `04_printer_queue.py` | Printer queue simulator |
| `05_customer_service_priority.py` | Customer service queue with VIP priority |
| `06_circular_queue_round_robin.py` | Circular queue for Round-Robin scheduling |
| `07_bfs_graph.py` | BFS (Breadth-First Search) using queue |
| `08_call_center.py` | Call center queue with wait time |
| `09_priority_queue_heapq.py` | Priority queue using heapq |
| `10_priority_queue_class.py` | Priority queue as class |
| `11_monotonic_queue.py` | Monotonic queue (sliding window) |
| `12_sliding_window_maximum.py` | Sliding Window Maximum (LeetCode 239) |
| `13_bfs_shortest_path.py` | BFS shortest path in unweighted graphs |
| `14_multi_source_bfs.py` | Multi-source BFS (rotting oranges pattern) |
| `15_task_scheduler_queue.py` | Task scheduler with cooldown |
