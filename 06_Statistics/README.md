# 06_Statistics

Fundamentos estadísticos para analistas.

**Requirements:** `numpy`, `scipy`

**Run:** `python 01_descriptive_stats.py` (or any `NN_*.py`) from `06_Statistics/`.

---

## Temas

- mean, median, mode
- variance, std
- percentiles
- probability basics
- distributions
- z-score
- correlation
- hypothesis basics

---

## Archivos

| File | Description |
|------|-------------|
| `01_descriptive_stats.py` | Mean, median, mode, variance, standard deviation |
| `02_percentiles_quantiles.py` | Percentiles, quantiles, IQR, quartiles |
| `03_distribution_basics.py` | Probability basics, normal, uniform, binomial |
| `04_correlation_covariance.py` | Correlation, covariance, Pearson |
| `05_zscore_outliers.py` | Z-score, outlier detection, standardization |
| `06_sampling_basics.py` | Random sample, sampling distribution, SE |
| `07_hypothesis_basics.py` | Hypothesis testing, p-value, t-test, confidence interval |
