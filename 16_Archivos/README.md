# 16_Archivos (Gestión de archivos)

Ejercicios para crear, leer, modificar, listar, verificar, eliminar, copiar y mover archivos en Python.

| Archivo | Contenido |
|---------|-----------|
| `archivo_01_crear_escribir.py` | Crear archivo y escribir con `open(..., "w")`, `write()`, `writelines()` |
| `archivo_02_leer.py` | Leer archivo: `read()`, `readline()`, `readlines()` e iterar por líneas |
| `archivo_03_anadir.py` | Añadir contenido al final con modo `"a"` (append) |
| `archivo_04_listar_directorio.py` | Listar archivos y carpetas: `Path.iterdir()`, `os.listdir()`, `glob()` |
| `archivo_05_verificar_existe.py` | Verificar si existe: `Path.exists()`, `is_file()`, `is_dir()` |
| `archivo_06_eliminar.py` | Eliminar archivo: `Path.unlink()`, `os.remove()` |
| `archivo_07_copiar_mover.py` | Copiar y mover: `shutil.copy2()`, `shutil.move()` |
| `archivo_08_with_contexto.py` | Usar `with open(...)` para cerrar el archivo automáticamente |
| `crud_empleados.py` | CRUD completo: crear, listar, ver, editar, eliminar registro y eliminar archivo (JSON) |

```bash
python 16_Archivos/archivo_01_crear_escribir.py
python 16_Archivos/archivo_02_leer.py
python 16_Archivos/archivo_03_anadir.py
python 16_Archivos/archivo_04_listar_directorio.py
python 16_Archivos/archivo_05_verificar_existe.py
python 16_Archivos/archivo_06_eliminar.py
python 16_Archivos/archivo_07_copiar_mover.py
python 16_Archivos/archivo_08_with_contexto.py
python 16_Archivos/crud_empleados.py
```
