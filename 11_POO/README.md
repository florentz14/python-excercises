# 11_POO (Programación Orientada a Objetos)

Ejercicios de **clases** en Python: herencia, polimorfismo y clases sencillas (vectores, matrices, datos).

## Clases sencillas (vectores, matrices, datos)

| Archivo | Contenido |
|---------|-----------|
| `clase_vector.py` | Vector (listas): add(), scale() |
| `clase_vector_numpy.py` | VectorNP (NumPy): add(), scale(), magnitude() |
| `clase_matrix.py` | Matrix (listas): add() |
| `clase_matrix_numpy.py` | MatrixNP (NumPy): add(), multiply(), transpose() |
| `clase_point2d.py` | Point2D (NumPy): move(), distance_to_origin() |
| `clase_dataset.py` | Dataset (NumPy): mean(), max(), normalize() |
| `clase_sparse_matrix.py` | SparseMatrix (dict): set(), get() |

## Clases con herencia y polimorfismo

| Archivo | Contenido |
|---------|-----------|
| `clase_animales.py` | Animal, Perro, Gato, Pájaro, Pez, León |
| `clase_dispositivos_moviles.py` | DispositivoMovil, Smartphone, Tablet, TeléfonoBasico, Smartwatch, EReader |
| `clase_personas.py` | Persona, Estudiante, Profesor, Ingeniero, Médico, Deportista |
| `clase_transporte.py` | Transporte, Automovil, Motocicleta, Bicicleta, Avión |
| `clase_usuarios.py` | Usuario, Cliente, Empleado, Vendedor, Gerente, Administrador, SoporteTecnico |
| `clase_usuarios_avanzada.py` | PasswordHasher, UsuarioAvanzado, GestorUsuarios (roles, sesiones, auditoría) |

Ejecutar desde la raíz del proyecto:

```bash
python 11_POO/clase_vector.py
python 11_POO/clase_vector_numpy.py
python 11_POO/clase_matrix_numpy.py
python 11_POO/clase_point2d.py
python 11_POO/clase_dataset.py
python 11_POO/clase_sparse_matrix.py
python 11_POO/clase_animales.py
```
