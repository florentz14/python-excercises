# Tipos de Datos Basicos

Ejercicios sobre tipos de datos fundamentales en Python: numeros complejos, variables, strings, comparacion, tipos y operaciones.

## Archivos (12)

| # | Archivo | Contenido |
|---|---------|-----------|
| 01 | `01_complex.py` | Numeros complejos, partes real/imaginaria, cmath |
| 02 | `02_variables.py` | int, float, variables, f-strings |
| 03 | `03_strings.py` | Strings, concatenacion, slicing, metodos |
| 04 | `04_comparacion.py` | Operadores ==, !=, <, >, <=, >= |
| 05 | `05_tipos.py` | type(), int(), float(), str(), bool() |
| 06 | `06_operaciones.py` | Aritmeticas (+, -, *, /, //, %), logicas (and, or, not) |
| 07 | `07_inputs.py` | input(), conversion de entrada a int/float |
| 08 | `08_conversion.py` | int(), float(), str(), bool(), list(), tuple(), set() |
| 09 | `09_constantes_asignacion.py` | Constantes, asignacion multiple, swap |
| 10 | `10_none.py` | Tipo None, is None |
| 11 | `11_print_sep_end.py` | print(sep=, end=) |
| 12 | `12_membresia.py` | Operadores in, not in |

## Ejecutar

```bash
cd tipos_datos
python 01_complex.py
python 03_strings.py
```

## Requisitos

- Python 3.x
- Modulo `cmath` (biblioteca estandar)
