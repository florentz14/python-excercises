# Data Ops App

Agrupa scripts `08-13` bajo estructura modular estilo `music_api`.

## Ejecutar

Desde `20_Databases`:

```bash
python -m data_ops_app.app.main
```

Directo por script:

```bash
python -m data_ops_app.app.main family_budget_loader
python -m data_ops_app.app.main tasks
python -m data_ops_app.app.main survey_load
python -m data_ops_app.app.main survey_filter_2002
python -m data_ops_app.app.main stocks
python -m data_ops_app.app.main customers
```
