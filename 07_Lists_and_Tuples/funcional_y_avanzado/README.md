# Programacion Funcional y Avanzada

Permutaciones, combinaciones, powerset, agrupacion por funcion, union/diferencia/interseccion con funciones personalizadas, y predicados (all, any).

## Archivos (11)

| # | Archivo | Contenido |
|---|---------|-----------|
| 01 | `01_generate_permutations.py` | Generar permutaciones |
| 02 | `02_generate_combinations.py` | Generar combinaciones |
| 03 | `03_all_combinations_elements.py` | Todas las combinaciones |
| 04 | `04_group_by_function_count.py` | Agrupar por funcion |
| 05 | `05_difference_with_function.py` | Diferencia con funcion |
| 06 | `06_common_with_function.py` | Comunes con funcion |
| 07 | `07_symmetric_difference_with_function.py` | Diferencia simetrica |
| 08 | `08_union_with_function.py` | Union con funcion |
| 09 | `09_all_true_for_function.py` | all() con predicado |
| 10 | `10_any_true_for_function.py` | any() con predicado |
| 11 | `11_powerset.py` | Conjunto potencia |

## Ejecutar

```bash
cd funcional_y_avanzado
python 01_generate_permutations.py
python 11_powerset.py
```

## Requisitos

- Python 3.x
- Modulo `itertools` (biblioteca estandar)
