# Ordenamiento y Organizacion

Ordenar listas por diferentes criterios: alfabetico, numerico, por longitud, por otra lista, shuffle y verificaciones.

## Archivos (10)

| # | Archivo | Contenido |
|---|---------|-----------|
| 01 | `01_sort.py` | Ordenar basico |
| 02 | `02_shuffle_list.py` | Mezclar lista |
| 03 | `03_sort_by_length_and_value.py` | Ordenar por longitud y valor |
| 04 | `04_check_list_sorted.py` | Verificar si esta ordenada |
| 05 | `05_sort_list_by_index.py` | Ordenar por indice |
| 06 | `06_sort_mixed_list.py` | Ordenar lista mixta |
| 07 | `07_sort_string_numbers_numerically.py` | Ordenar numeros como numeros |
| 08 | `08_sort_digits_ascending_descending.py` | Ordenar digitos asc/desc |
| 09 | `09_sort_by_another_list.py` | Ordenar por otra lista |
| 10 | `10_shuffle_new_list.py` | Mezclar en nueva lista |

## Ejecutar

```bash
cd ordenamiento
python 01_sort.py
python 09_sort_by_another_list.py
```

## Requisitos

- Python 3.x
- Modulo `random` para shuffle (biblioteca estandar)
