### Extras: práctica adicional (280 ejercicios)

Archivos `extra_001_*.py` … `extra_280_*.py` en secuencia numérica.

- **001–081**: sumas, máximos/mínimos, duplicados, ordenar, flatten, permutaciones, etc.
- **082–200**: combinaciones, matrices, listas anidadas, columnas, rotar, frecuencia, etc.
- **201–280**: filtros, chunk, índices, mapeo, flatten profundo, potencias, progresiones, etc.

Para seguir la secuencia, ordena los archivos por nombre; la numeración con tres dígitos (001–280) mantiene el orden correcto.
