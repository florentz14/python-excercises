# Ejercicios Generales

Manipulacion general de listas: modificar, insertar, eliminar, rotar, intercalar, run-length encoding, conversion de tipos, iteracion ciclica, bigrams, fibonacci y ejercicios variados.

## Archivos (133)

Ejercicios numerados del 014 al 133 que cubren:

- **Manipulacion**: insertar, eliminar, rotar, mover elementos
- **Conversion**: string a lista, lista a dict, tipos de datos
- **Operaciones**: concatenar, dividir, intercalar, chunk
- **Algoritmos**: run-length, fibonacci, merge sorted
- **Estadisticas**: max, min, promedios, gaps
- **Estructuras**: listas de dicts, listas de listas
