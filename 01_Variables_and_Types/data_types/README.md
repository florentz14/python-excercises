# data_types/ - Basic Data Types

Exercises on fundamental data types in Python: complex numbers, variables, strings, comparison, and types.

## Files

| # | File | Content |
|---|------|---------|
| 01 | `01_complex.py` | Complex numbers, real/imaginary parts, cmath |
| 02 | `02_variables.py` | int, float, variables, f-strings |
| 03 | `03_strings.py` | Strings, concatenation, slicing, methods |
| 04 | `04_comparison.py` | Operators ==, !=, <, >, <=, >= |
| 05 | `05_types.py` | type(), int(), float(), str(), bool() |
| 07 | `07_inputs.py` | input(), convert input to int/float |
| 08 | `08_conversion.py` | int(), float(), str(), bool(), list(), tuple(), set() |
| 09 | `09_constants_assignment.py` | Constants, multiple assignment, swap |
| 10 | `10_none.py` | None type, is None |
| 11 | `11_print_sep_end.py` | print(sep=, end=) |
| 12 | `12_membership.py` | Operators in, not in |
| 13 | `13_logical.py` | and, or, not, short-circuit, combination with comparisons |
| 14 | `14_control_flow.py` | if/elif/else, for, while |
| 15 | `15_hello_world_greet.py` | Hello World, greet function |
| 16 | `16_table_3_column_numbers.py` | 3-column number table |
| 17 | `17_table_fstring_alignment.py` | f-string alignment |
| 18 | `18_table_3_column_fruits.py` | 3-column fruit table |
| 19 | `19_table_headings_prices.py` | Table with headings and prices |
| 20 | `20_table_print_end_parameter.py` | print with end parameter |
| 21 | `21_table_float_precision.py` | Float precision formatting |
| 22 | `22_hello_print_basics.py` | Basic print, separators |
| 23 | `23_circle_area_variables.py` | Circle area with variables |
| 24 | `24_arithmetic_operators.py` | +, -, *, /, % |
| 25 | `25_variable_swapping.py` | Swapping variables, +=, *= |
| 26 | `26_integer_division.py` | // operator |
| 27 | `27_modulo_operator.py` | % operator |
| 28 | `28_augmented_assignment.py` | +=, -=, *=, /= |
| 29 | `29_swapping_example.py` | Swapping example (shoes, hats) |
| 30 | `30_input_type_conversion.py` | input(), int(), float() conversion |

## Run

```bash
cd data_types
python 01_complex.py
python 03_strings.py
```

## Requirements

- Python 3.x
- Module `cmath` (standard library)
