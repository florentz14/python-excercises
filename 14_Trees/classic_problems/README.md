Professional list used in **Google, Amazon, Meta, Microsoft** interviews. Organized by concept and difficulty.

## Pattern Recognition (90% of tree problems)

Before solving, ask:
1. **DFS or BFS?**
2. **Recursion?**
3. **BST property?** (Left < Root < Right)
4. **Path problem?**
