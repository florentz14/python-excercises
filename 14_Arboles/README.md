# 14_Árboles (Trees)

Estructuras de **árbol binario**: nodos con hijo izquierdo y derecho.

| Archivo | Contenido |
|---------|-----------|
| `arbol_01_nodo.py` | Clase Nodo y construcción de un árbol |
| `arbol_02_recorrido.py` | Recorridos inorden, preorden, postorden |
| `arbol_03_altura.py` | Altura y tamaño (número de nodos) |

```bash
python 14_Arboles/arbol_01_nodo.py
python 14_Arboles/arbol_02_recorrido.py
python 14_Arboles/arbol_03_altura.py
```
